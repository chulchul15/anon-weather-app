# 🌦 Weather App

A simple weather app UI using HTML, CSS, and JavaScript.

## 🚀 Features
- Search city weather
- Responsive layout
- Clean modern UI
- Interactive card animation

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download project files
2. Open `index.html`
3. Type a city and click search

## 🌐 Future Improvements
- Real weather API
- Dynamic weather icons
- Dark/light theme toggle
